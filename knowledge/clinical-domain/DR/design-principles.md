# DR UX Design Principles


## 1. Support Clinical Workflow

Design follows radiographer workflow.


## 2. Preserve Safety Authority

AI recommends.

Human decides.


## 3. Reduce Repeat Exposure

Prevent errors before exposure.


## 4. Make System State Visible

The user should understand:

- What happened?
- What is happening?
- What should happen next?


## 5. Avoid Hidden Automation

Automation should be explainable.
