# Digital Radiography (DR) Domain Knowledge


## Purpose

This section provides domain knowledge about digital radiography systems.

The goal is to help AI understand:

- DR clinical workflow
- radiographer tasks
- equipment operation
- safety considerations
- UX design implications


## Scope

Including:

- Fixed DR systems
- Mobile DR systems
- High-end robotic DR systems
- Emergency radiography workflow


## Users

Primary user:

- Radiographer / Radiologic Technologist


Secondary users:

- Service engineer
- Clinical educator
- Radiologist


## Design Principle

DR UX design should support:

- Fast examination workflow
- Accurate patient positioning
- Reduced repeat exposure
- Clear safety feedback

AI should assist human decision-making,
not replace professional judgement.
